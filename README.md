# Binary-Tree
Este repositório contém a implementação de uma Árvore Binária, desenvolvida como parte de um trabalho acadêmico na disciplina de Estruturas de Dados. O projeto tem como objetivo aplicar, de forma prática, os conceitos fundamentais relacionados às árvores binárias, uma das estruturas de dados mais importantes da computação.

A implementação foi realizada com foco em organização, clareza, correção algorítmica e boas práticas de programação, permitindo compreender o funcionamento interno da estrutura, bem como suas principais operações.

## Objetivos do Trabalho
- Compreender o funcionamento de uma Árvore Binária
- Implementar a estrutura de forma computacional
- Aplicar conceitos de Programação Orientada a Objetos
- Desenvolver raciocínio lógico e algorítmico
- Consolidar o aprendizado teórico por meio da prática

## Conceitos Abordados
O projeto aborda os seguintes conceitos fundamentais:

- Estrutura de Árvore Binária
- Nós (nodos) e suas referências
- Relação entre raiz, filhos e subárvores
- Inserção de elementos

### Percursos em árvores:

- Pré-ordem
- Em ordem
- Pós-ordem
- Organização hierárquica dos dados
- Uso eficiente de ponteiros/referências

## Funcionalidades Implementadas
O sistema permite:

- Criar uma árvore binária
- Inserir novos valores na árvore
- Percorrer a árvore utilizando diferentes tipos de percurso
- Exibir os elementos de acordo com o percurso escolhido
- Demonstrar o comportamento estrutural da árvore binária
